This page provides material property data for concrete, steel, and stainless steel used in anchor channel calculations per EN 1992-4.

**Key:**
- f<sub>ck</sub> = Characteristic cylinder strength
- f<sub>ck,cube</sub> = Characteristic cube strength
- f<sub>cm</sub> = Mean cylinder strength
- f<sub>ctm</sub> = Mean tensile strength
- E<sub>cm</sub> = Mean elastic modulus

### Values Used in EN 1992-4

For anchor channel calculations, the key parameter is f<sub>ck</sub>:

```
Concrete cone: NRk,c ∝ √fck × hef^1.5
Edge failure:  VRk,c ∝ √fck × c1^1.5
```

<Note>
  Higher concrete strength increases capacity, but benefits diminish
  for f<sub>ck</sub> > 50 MPa. Other factors often govern.
</Note>

```
PREN = %Cr + 3.3×%Mo + 16×%N
```

## Design Value Calculations

### Steel Resistance

```
NRd,s = As × fuk / γMs
      = As × fuk / 1.2

VRd,s = 0.6 × As × fuk / γMs
      = 0.6 × As × fuk / 1.0
```

### Concrete Cone Resistance

```
NRd,c = k1 × √fck × hef^1.5 × ψ factors / γMc
      = k1 × √fck × hef^1.5 × ψ factors / 1.5
```
